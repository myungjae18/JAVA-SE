아스키코드 : 1바이트 크기를 갖는 키코드 체계
유니코드 : 2바이트 크기를 갖는 키보드 체계
javaSE : 자바로 개발할 수 있는 플랫폼 중 독립실행형 응용프로그램을 개발할 수 있는 플랫폼
javaEE : 자바로 개발할 수 있는 분야 중 웹어플리케이션을 개발할 수 있는 플랫폼
javaME : 자바로 개발할 수 있는 분야 중 임베디드 응용프로그램을 개발할 수 있는 플랫폼
JVM : 자바가상머신
stack : 메모리의 데이터 처리 방법 중 FILO 방식
heap : JVM의 메모리 구조에서 인스턴스가 생성되는 영역
gabage collector : JVM에 생성된 인스턴스 제거자
클래스 변수 : static 수식자로 선언된 변수
인스턴스 변수 : 인스턴스에 소속되는 멤버변수
레퍼런스 변수 : 인스턴스의 주소값을 담는 변수
static 초기화 블럭 : 실행 직전에 호출되는 초기화 블럭
인스턴스 초기화 블럭 : 인스턴스 생성 시 호출되는 초기화 블럭
컴파일 : 원본소스를 기계어로 변환하는 과정
classpath : 환경변수 중 클래스의 경로를 담는 변수
path : 환경변수 중 일반 실행파일의 경로를 담는 변수
-d 옵션 : 컴파일 명령의 옵션 중 컴파일 결과물의 위치를 지정할 수 있는 옵션
package : 중복을 방지하고 관련성 있는 클래스끼리 묶어놓은 디렉토리
import : 사용하려는 클래스의 위치를 지정하기 위한 키워드
OOP : 객체 지향 언어
상속 : 예전에 정의된 클래스의 코드를 재사용하기 위한 객체지향의 코드 작성 기법
static : 클래스 소속의 변수와 메서드를 지정하기 위한 수식자
생성자 : 객체 생성 시 해당 객체를 초기화하기 위한 메서드
OverLoading : 하나의 클래스 내에서 메서드명을 중첩 정의할 수 있는 메서드 정의 기법
OverRiding : 상속관계에서 부모의 메서드를 자식이 재정의할 수 있는 기법
은닉화(캡슐화) : 데이터인 변수는 private으로 접근을 막고, 메서드를 통해 데이터를 정의할 수 있도록 허용하는 OOP 클래스의 정의 방법
디폴트 생성자 : 개발자가 생성자를 정의하지 않으면 컴파일러에 의해 정의되는 생성자
추상 클래스 : 불완전한 클래스
인터페이스 : 추상 메서드만을 보유한 기능 집합
SQL : 구조화된 질의어, DBMS에서만 사용가능한 언어
SQLPlus : oracle에서 제공하는 접속 클라이언트 프로그램
SYS : 오라클의 최고 권한을 갖는 관리자
SYSTEM : SYS 다음으로 관리자 권한을 갖는 사용자
서브쿼리 : 쿼리문 안의 쿼리
avg() 집계함수 : 평균을 구하는 집계함수
count() 집계함수 : 레코드의 갯수를 구하는 집계함수
DDL : 데이터베이스 정의어
DML : 데이터베이스 조작어
DCL : 데이터베이스 제어어
Tablespace : 오라클에서 사용하는 데이터베이스 단위
DBF 파일 : 오라클의 데이터베이스 파일 확장자
services.msc : 서비스 창을 띄우는 단축 명령어
DROP 문 : DDL 중 삭제 쿼리
ALTER 문 : DDL 중 변경 쿼리
Delete 문 : DML 중 레코드 삭제 쿼리
Insert 문 : DML 중 레코드 입력 쿼리
where 조건문 : select 쿼리의 조건문
order by : 정렬 키워드
DESC : 내림차순 정렬
conn : SQLPlus의 현재 접속에서 다른 접속으로 변경
제약조건 : 데이터 무결성을 위한 컬럼에 부여하는 조건
primary key : 제약조건 중 not null + unique
unique : 중복을 허용하지 않는 제약조건
default : 기본값을 명시할 수 있는 제약조건
not null : null을 허용하지 않는 제약조건
foreign key : 외래키 제약 조건
ANSI Query : ANSI에서 지정한 표준쿼리로써 모든 DBMS에서 공통으로 사용 가능
join : 정규화에 의해 물리적으로 분리된 테이블을 마치 하나의 테이블처럼 보여주는 select
시퀀스(sequence) : 숫자 증가값만을 보유한 테이블
auto_increment : mysql(=mariadb)에서 컬럼에 부여되는 자동 증가 속성
sysdate : oracle의 현재 날짜를 구하는 함수
timestamp : mysql의 날짜 자료형
now() :  mysql의 현재 날짜를 구하는 함수
public : 접근제한자 중 공개접근 제한자
protected : 접근제한자 중 같은 패키지나 상속관계의 객체에게만 접근을 허용하는 접근제한자
default : 개발자가 아무것도 명시하지 않으면 적용되는 접근제한자로써, 같은 패키지 내의 객체에게만 접근을 허용함
private : 자기 자신만 접근을 허용함
강제 형변환 : cast 연산자에 의한 형변환
자동 형변환 : 시스템에 의해 자동으로 일어나는 형변환
cast 연산자 : 형변환 시 사용되는 연산자
demotion : 기본 자료형 간 형변환 중 큰 자료형에서 작은 자료형으로의 형변환
promotion : 기본 자료형 간 형변환 중 작은 자료형에서 큰 자료형으로의 형변환
UpCasting : 객체 간 형변환 중 작은 자료형에서 큰 자료형으로의 형변환
DownCasting : 객체 간 형변환 중 큰 자료형에서 작은 자료형으로의 형변환
추상 메서드 : 불완전한 메서드
API Document : 응용프로그램의 사용법에 대한 설명서
Constant Pool : String 객체의 중복생성을 방지하기 위한 메모리 영역
명시적 생성법 : new 연산자에 의해 String 객체를 생성하는 방법
묵시적 생성법 : 기본 자료형처럼 String 객체를 생성하는 방법
Call by value : 값에 의한 호출(값 자체가 전달되는 것이 아니라 그 값과 동일한 값이 전달)
Call by Reference : 주소값에 의한 호출
equals() 메서드 : 최상위 객체인 Object 클래스가 보유한 메서드로서, 객체의 내용을 비교하는 메서드
Object 클래스 : 자바의 최상위 클래스
Wrapper 클래스 : 기본자료형과 객체자료형 간의 형변환을 지원하는 클래스로써, 자바의 기본자료형마다 1:1대응하는 래퍼클래스가 있다.
ex) int --> Integer, char --> Character
.class 파일 : 컴파일 결과물로 생성되는 바이너리 파일
Polymorphism(다형성) : 상위 객체로 하위의 다양한 객체의 메서드를 호출할 수 있는 OOP 클래스 정의 기법 중 하나
has a 관계 : 부품관계를 의미하며, 특정 클래스가 특정 클래스를 멤버변수로 보유한 관계
is a 관계 : 상속관계를 의미하며, 특정 클래스가 특정 클래스를 부모로 둔 관계
rt.jar : sun사에서 제공하는 개발에 필요한 API 객체들이 압축되어 있는 파일
상수 : 프로그램이 종료될 때 까지 그 값이 변경되지 못하는 데이터
super : 부모 클래스를 가리키는 레퍼런스 변수
extends : 상속 구현 시 사용되는 키워드
implements : 인터페이스 구현 시 사용되는 키워드
abstract 수식자 : 수식자 중 클래스와 메서드에 적용할 수 있는 수식자로써 클래스에 붙이면 추상클래스, 메서드에 붙이면 추상메서드를 선언하게 된다
final 수식자 : 변수에 붙이면 더 이상 변하지 않게 하고, 메서드에 붙이면 오버라이드를 허용하지 않고 클래스에 붙이면 상속을 허용하지 않는 수식자
split() 메서드 : String 클래스의 메서드 중 문자열 내의 토큰을 기준으로 문자열을 분리시켜 배열로 반환하는 메서드
javadoc : API Document 생성 명령어
